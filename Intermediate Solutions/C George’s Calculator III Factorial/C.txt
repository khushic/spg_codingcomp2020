George is still going strong with his calculator functions. The next one he wants to add is one that finds the factorial of a given number. The factorial of integer n, written as n! is equal to n*(n-1)*(n-2)...*2*1.
Input: int n
Precondition: 0 < n < 20
Output: int
Sample:
Input: 5					Output: 120
Explanation: 5! = 5*4*3*2*1 = 120
